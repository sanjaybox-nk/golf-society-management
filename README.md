# Golf Society Management (BoxyArt)

A premium, hardened Flutter application for managing golf societies, featuring a high-contrast design system, dynamic notifications, and robust administrative controls.

## ✨ Key Features
- **Dynamic Dashboard**: Real-time notifications and upcoming match tracking.
- **Communications Hub**: Admin-led broadcasting with custom distribution lists.
- **Member Management**: Searchable directory with photo uploads and role assignment.
- **Events Hub**: Full event lifecycle management from registration to results.
- **BoxyArt UI**: A custom, hardened design system for a premium feel.

## 🛠 Tech Stack
- **Framework**: Flutter (Dart)
- **State Management**: Riverpod (with Code Generation)
- **Navigation**: GoRouter (Shell Routes)
- **Database/Auth**: Firebase (Integration in progress)

## 🚀 Getting Started
1. Clone the repository.
2. Run `flutter pub get`.
3. Run `flutter pub run build_runner build` to generate models.
4. Launch with `flutter run`.
